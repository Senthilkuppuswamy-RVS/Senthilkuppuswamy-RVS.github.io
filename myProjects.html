PACMEN EXERCISE

This Project creates PacMen on demand.
These PacMen are made to move across the Page.
When we click the "Add PacMan" button, a new pacman appears at a random position on the screen.
When we click the "Start Game" button, the pacman or pacmen created starts moving at random velocity on the screen.

EYE EXERCISE

This Project have two eyes which follows the mouse movement.
Both the eyes are synchronised to follow the directions of mouse pointer.
It looks like a person following the mouse.
The mouse movement event is used to update the position of the "eye" element displayed on the page.
Mouse events are a popular way to bind the DOM elements to events.

REAL TIME BUS TRACKER

This Project uses MBTA data to determine the stops between Harvard and MIT.
Mapbox Account was created and the access token is used to the mapboxgl instance.
The marker is created and added to the map using mapboxgl.
The function move is used to move the marker from MIT to Harvard.
The animated marker highlights the bus routes on the map.
